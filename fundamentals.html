Use dojo for an anchor with an id <a id="a-tag-dojo" href="http://google.com">Use dojo for an anchor with an id</a>

<script>
dojo.query('#a-tag-dojo').connect('click', function (event) {
alert("dojo!");
dojo.stopEvent(event);
});
</script>

Use dojo for an anchor with a class <a class="dojo" href="http://google.com">Use dojo for an anchor with a class</a>

<script>
dojo.query('a.dojo').connect('click', function (event) {
alert("dojo!");
dojo.stopEvent(event);
});
</script>

<script>

<script>

</script>

Stuff to notice

  • You can do the same thing in lots of ways.
  • There's a big difference between an error and a bug. If you check your console, you won't see any errors, but the code in the third row of the table has a bug.
  • Dojo and jquery make it easy to ignore that different browsers work differently.
  • Starting to use dojo and jquery is very easy -- just add a script tag to the top of your page and you're ready to go. This may not even increase page load time, if your readers have already downloaded it while on another site.
